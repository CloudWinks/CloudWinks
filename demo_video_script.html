CloudWinks Demo Video Script

3-5 minute demonstration for Google OAuth verification

Video Overview

Duration: 3-5 minutes

Purpose: Demonstrate how CloudWinks uses Google OAuth scopes for enhanced functionality

Target Audience: Google verification team

Video Structure

1. Introduction (30 seconds)

Visual: App logo and title screen

"Welcome to CloudWinks, a comprehensive business management platform that integrates seamlessly with Google services to enhance collaboration and productivity."

2. User Authentication (45 seconds)

Visual:

  • Show Google login button
  • Demonstrate login process
  • Show user profile with Google data
"Users can securely authenticate using their Google account, providing basic profile information for seamless account creation and management."

3. Google Contacts Integration (1 minute)

Visual:

  • Show user search feature
  • Demonstrate contact lookup from Google Contacts
  • Show contact import process
  • Display contact-based user suggestions
"When users need to collaborate with team members, CloudWinks can search their Google Contacts to quickly find and add existing contacts. This eliminates manual data entry and ensures accurate contact information."

Key Points:

  • Read-only access to contacts
  • User-initiated search only
  • No contact modification
  • Clear user consent

4. Google Drive Integration (1.5 minutes)

Visual:

  • Show file attachment feature
  • Demonstrate Google Drive file picker
  • Show file preview and metadata
  • Display file organization features
"For file management, CloudWinks integrates with Google Drive to allow users to attach existing files to projects and tasks. Users can browse their Drive, select files, and attach them without downloading or duplicating storage."

Key Points:

  • File browsing and selection
  • Metadata access for efficient browsing
  • User-controlled file access
  • No unauthorized file access

5. Advanced Features (45 seconds)

Visual:

  • Show project creation with file attachments
  • Demonstrate team collaboration features
  • Display data export to Google Drive
"These integrations enable powerful collaboration features like project file management, team member suggestions, and automated data exports to Google Drive."

6. User Control and Privacy (30 seconds)

Visual:

  • Show account settings
  • Demonstrate scope management
  • Display privacy controls
"Users maintain full control over their data. They can revoke access to any Google service at any time, review what data we access, and export or delete their information."

7. Conclusion (30 seconds)

Visual: App interface with all features working together

"CloudWinks enhances productivity by integrating with users' existing Google services while maintaining the highest standards of privacy and user control."

Technical Requirements

Video Specifications

  • Resolution: 1920x1080 (Full HD)
  • Format: MP4
  • Duration: 3-5 minutes
  • Audio: Clear narration with background music
  • Subtitles: Optional but recommended

Recording Guidelines

  • Use Real Data: Show actual app functionality with test data
  • Clear Navigation: Make all user actions visible and understandable
  • Professional Presentation: Clean interface, no development artifacts
  • Consent Indicators: Show where users grant permissions
  • Error Handling: Demonstrate graceful handling of permission denials

Key Scenes to Include

Scene 1: Login Process

  • Show Google OAuth consent screen
  • Highlight requested scopes
  • Demonstrate successful authentication

Scene 2: Contact Search

  • Show empty user search field
  • Demonstrate Google Contacts integration
  • Show contact selection and import

Scene 3: File Management

  • Show file attachment interface
  • Demonstrate Google Drive picker
  • Show file preview and metadata

Scene 4: User Controls

  • Show account settings page
  • Demonstrate scope management
  • Show privacy and data controls

Success Metrics

The video should clearly demonstrate:

  • ✅ Clear user benefit for each scope
  • ✅ Transparent permission requests
  • ✅ User control over data access
  • ✅ Professional implementation
  • ✅ Privacy-conscious design
